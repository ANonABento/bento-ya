# T030: Metrics Dashboard

## Summary

Analytics view showing productivity metrics: tasks completed, time per stage, agent success rates, cost tracking, time saved estimates.

## Acceptance Criteria

- [ ] Dashboard accessible from workspace header or settings
- [ ] Tasks completed per day/week (bar chart)
- [ ] Average time per pipeline stage (horizontal bar chart)
- [ ] Agent success rate by column (percentage)
- [ ] Cost tracking: API usage per task, per workspace, per model (rolling 24h/7d/30d)
- [ ] Time saved estimates (comparing agent duration vs manual estimate)
- [ ] Date range selector
- [ ] Export data as CSV
