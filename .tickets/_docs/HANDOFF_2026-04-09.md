## Priority Task: Rewrite Terminal Bridge Layer

### What to fix:
1. **Single managed bridge per task** — currently bridges can duplicate (mpsc + broadcast). Need one bridge with cancellation.
2. **Shell ready detection** — replace 200ms fixed delay with prompt regex matching before writing trigger commands.
3. **Trigger PATH resolution** — `codex`/`claude` not found from Tauri process env. Need to resolve CLI paths or inject into PTY env.
4. **Detailed error propagation** — "Execution failed" is too generic. Surface actual error from bridge.rs to the task's pipeline_error field.

### Architecture (see HANDOFF_TERMINAL_REWRITE.md for full details):
```
PtyTransport (read thread)
  → mpsc → async buffer task → broadcast sender (ONLY external interface)
                                    ↓
                              ManagedBridge {handle, cancel_token}
                                    ↓
                              app.emit("pty:{id}:output")
```
