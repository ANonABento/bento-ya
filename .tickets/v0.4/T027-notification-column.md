# T027: Notification Column

## Summary

A column that reminds users to notify stakeholders after a task is complete and deployed. Shows context of what changed.

## Acceptance Criteria

- [ ] Default "Notify" column template available
- [ ] Shows task summary, changes, and deployment status
- [ ] Manual exit criteria: user confirms notification was sent
- [ ] Optional: integrate with Slack/email (P2, just placeholder for now)
- [ ] Task detail shows who to notify (configurable per task or per workspace)
